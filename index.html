Assignment Instructions

Reference Image: Download the reference image from the link below to use as your base for drawing:

Download Reference Image

  1. Create a geodatabase and define the WGS_1984_Web_Mercator coordinate system.
  2. Create the following layers:
    • Agricultural
    • Urban Land
    • Streets
    • Landmarks
    • Vacant
    • Water Area
  3. Define attributes for each layer as follows:
    • Agricultural: Street name, Governorate name,Type (Subtype: Park, Garden, Natural Reserve,Agricultur Land), Tax value (0-10,000).
    • Urban Land: Zone ID (Unique Identifier),Population Density (Domain: Low, Medium, High), Zone Type (Subtype: Residential, Mixed-use, Rural), Street name, Governorate name, Price (0-1,000,000), Number of floors, etc.
    • Streets: Road Type (Subtype: Highway, Main, Secondary, Local),Length (Meters) , Road ID (Unique Identifier),Street name, Governorate name.
    • Landmarks: Landmark ID,Type (Pharmacy, Mall, Mosque, etc.), Place name, Street name.
    • Vacant: Type , Governorate Name (Alexandria).
  4. Draw the phenomena in the study area and add dummy data to each layer.
  5. Activate tracking for all layers.
  6. Set unique symbology for:
    • Urban Land: Use unique values based on building type.
    • Landmarks: Assign symbols from a predefined gallery.
  7. Create a professional map layout with the following elements:
    • North Arrow
    • Coordinate System
    • Data Source
    • Scale Bar
    • Legend
    • Layout Date and Data Date
    • Author
    • Logo
    • Locator Map
    • Table listing top five regions by building price
  8. Export the outputs as follows:
    • Map as JPG and PDF.
    • Streets layer as KML file.
    • Buildings layer as Excel file.
  9. Deliverables should be organized in a folder containing all required files.

Expected Outputs

  • JPG and PDF files for the map layout.
  • KML file for the Streets layer.
  • Excel file for the Buildings layer.
  • GDB and APRX project files.
